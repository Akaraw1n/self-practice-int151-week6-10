สัปดาห์นี้ได้นำ practice จากในห้องมาทำต่อ โดยนำเรื่อง event มาใช้ดังตัวอย่าง 
คือ 1. click เป็น event ที่เกิดจากการที่เรา click mouse 
2.keydown เป็น event ที่เกิดตอนที่เรา typing 
3.enter เป็น event ที่เกิดตอนที่เรากดปุ่ม Enter
และทบทวนเรื่องการเลือก query ให้ตรงกับจุดที่เราต้องการให้เกิด event 
โดยการใช้ element และ id มาใช้ในการ query และการนำ condition เพื่อให้เช็ดผลลัพธ์ที่ต่างกัน
